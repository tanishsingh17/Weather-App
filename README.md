# 🌤️ Weather Web Application

A dynamic Weather Web Application built using **Python (Flask)** that allows users to select a country and city to view real-time weather information using the OpenWeatherMap API.

---

## 🚀 Features

* 🌍 Dynamic Country → City dropdown
* ☁️ Real-time weather data
* 🌡️ Temperature, humidity, wind speed, and weather conditions
* ⚡ Optimized request flow using POST → Redirect → GET
* 🛡️ Error handling for invalid inputs and API issues
* 📱 Clean and responsive UI

---

## 🛠️ Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript
* OpenWeatherMap API
* JSON
* python-dotenv
